# Data Handling

## Data Storage

Cotiza CPQ stores customer data natively within the customer's Salesforce org.

## External Data Transmission

Customer quote and pricing data is not transmitted to external servers unless explicitly configured by the customer.

## Authentication

Authentication is managed through Salesforce authentication and permission controls.

## Security Model

Cotiza CPQ respects:

- Salesforce CRUD permissions
- Field-level security
- Sharing rules
- Role hierarchy access
