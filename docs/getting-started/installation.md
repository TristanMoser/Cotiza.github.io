# Installation

## Prerequisites

Before installing Cotiza CPQ:

- Salesforce Enterprise Edition or higher
- System Administrator access
- API Enabled permission
- Lightning Experience enabled

## Installation Steps

1. Open the installation link
2. Select Install for Admins Only
3. Approve third-party access if prompted
4. Wait for package installation to complete

## Post Install

After installation:

1. Assign permission sets
2. Configure products and price books
3. Configure quote templates
4. Verify user access

## Estimated Installation Time

15–30 minutes
