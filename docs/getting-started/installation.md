# Installation

## Prerequisites

Before installing Cotiza CPQ:

- Salesforce Professional Edition or higher
- Enabled access to the standard Salesforce Quote object in Quote Settings in Salesforce Setup
- Lightning Experience enabled
- System Administrator access

## Installation Steps

1. Open the installation link
2. Select Install for Admins Only
3. Wait for package installation to complete

## Post Install

After installation:

1. Assign permission sets
2. Configure products and price books
3. Create Cotiza System Settings record
4. Create your first Cotiza Playbook

## Estimated Installation Time

15–30 minutes
