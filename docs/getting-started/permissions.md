# Permissions

## Permission Sets

Cotiza CPQ includes the following permission sets:

| Permission Set | Purpose |
|---|---|
| Cotiza Admin | Full administrative access |
| Cotiza Sales User | Standard quote creation access |
| Cotiza Approver | Approval workflow access |

## Required Salesforce Permissions

Users should have:

- API Enabled
- Run Reports
- Create and Edit on Quote objects
- Access to related Product objects
