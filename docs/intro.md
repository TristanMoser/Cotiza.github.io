# Cotiza CPQ

Cotiza CPQ is a native Salesforce CPQ solution focused on fast, flexible quote generation and streamlined sales workflows.

## Key Features

- Native Salesforce architecture
- Quote and proposal generation
- Flexible pricing configuration
- Approval workflows
- Product and pricing management
- PDF generation
- Lightning Experience support

## Supported Salesforce Editions

- Enterprise Edition
- Unlimited Edition
- Performance Edition
